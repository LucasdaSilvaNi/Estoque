O sistema SAM foi projetado para atender desde pequenos almoxarifados até estruturas com almoxarifados distribuídos, totalmente em ambiente web, possibilitando o controle de cada almoxarifado e o gerenciamento centralizado dos mesmos.

O SAM utiliza a estrutura de codificação de Materiais utilizada pelo Siafísico, ou seja Grupo / Classe / Material / Item de Material. Além disso, o SAM opera integrado com os sistemas Siafem e Siafísico.
